# zsh

## Setup

- Install oh-my-zsh ((instructions)[https://github.com/ohmyzsh/ohmyzsh])
- Symlink dotfiles
```sh
  ln -s ~/Projects/.dotfiles/zsh/.zshrc ~/.zshrc
  ln -s ~/Projects/.dotfiles/zsh/.zprofile ~/.zprofile
```
