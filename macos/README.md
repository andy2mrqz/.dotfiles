# macos installation

## Setup

- install [ghostty](https://ghostty.org/download)
- install homebrew, then use the brew/Brewfile to install all packages
- neovim (built from source):
  `git clone https://github.com/neovim/neovim ~/Projects/neovim` then run
  `update-neovim.sh -f` to build and install
- install mise
- install
  [claude code](https://code.claude.com/docs/en/quickstart#step-1-install-claude-code)

### other

- setup new ssh key, [add to github](https://github.com/settings/keys)
- get bitwarden extension setup
